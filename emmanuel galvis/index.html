PROGRAMACION

La Programación es el proceso de crear instrucciones que una computadora puede entender y ejecutar para realizar tareas específicas. En otras palabras, es “escribir órdenes” usando un lenguaje de programación para que una máquina pueda resolver problemas o hacer funciones automáticas.

HTML, JAVASCRIPT Y PYTHON

HTML es el lenguaje que se usa para crear la estructura de una página web, como textos, imágenes y botones. JavaScript es el que le da interactividad a la página, permitiendo que funcionen cosas como botones, animaciones o formularios. Python es un lenguaje de programación fácil de aprender que se usa para crear programas, páginas web, inteligencia artificial y automatizar tareas.

FUNCIONES

Las funciones en programación son bloques de código que realizan una tarea específica y se pueden usar varias veces sin volver a escribir todo. Sirven para organizar mejor el programa, evitar repetir código y hacerlo más fácil de entender. Por ejemplo, una función puede sumar dos números, mostrar un mensaje o calcular un resultado cuando se le llama.

CONCEPTOS DE PROGRAMACIÓN

Algunos conceptos básicos que he aprendido en programación.

parseInt()

Convierte un texto o número decimal en un número entero. Por ejemplo, "25.8" se convierte en 25.

parseFloat()

Convierte un texto en un número decimal. Por ejemplo, "25.8" se convierte en 25.8.

Ciclo While

Es una estructura repetitiva que ejecuta instrucciones mientras una condición sea verdadera.

Switch Case

Permite elegir entre varias opciones dependiendo del valor de una variable.
