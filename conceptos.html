Conceptos Aprendidos

Ciclo For

El ciclo for es una estructura repetitiva utilizada en programación para ejecutar instrucciones varias veces automáticamente. Se utiliza cuando conocemos la cantidad de veces que queremos repetir una acción. Es muy útil para recorrer listas, mostrar números, realizar cálculos y automatizar procesos.

Funciones

Las funciones son bloques de código organizados que realizan tareas específicas. Ayudan a reutilizar instrucciones, mejorar el orden del programa y facilitar la lectura del código.

Git

Git es un sistema de control de versiones que permite guardar cambios realizados en proyectos de programación. También ayuda a trabajar de forma organizada y recuperar versiones anteriores.

GitHub

GitHub es una plataforma online utilizada para almacenar proyectos creados con Git. Permite compartir código, trabajar en equipo y publicar proyectos web.

Lenguajes de Programación

Los lenguajes de programación permiten crear aplicaciones, videojuegos, páginas web y programas informáticos. Algunos ejemplos son: HTML, CSS, JavaScript y Python.

parseInt()

La función parseInt() convierte un texto o número decimal en un número entero. Es muy útil para trabajar con datos numéricos ingresados por usuarios.

parseFloat()

parseFloat() permite convertir textos en números decimales. Se utiliza en cálculos matemáticos, operaciones con precios y datos que necesitan decimales.

Switch Case

Switch Case permite ejecutar diferentes acciones dependiendo del valor de una variable. Es una alternativa más organizada cuando existen muchas condiciones.
