Conceptos de Programación

Una colección de temas importantes que he aprendido durante mi proceso en el mundo del desarrollo y la programación.

Temas Aprendidos

Ciclo For

El ciclo for permite repetir instrucciones automáticamente. Es muy útil para contar números, recorrer listas y automatizar procesos.

Funciones

Las funciones ayudan a organizar el código y reutilizar tareas específicas dentro de un programa.

Git

Git es un sistema de control de versiones utilizado para guardar cambios y trabajar de forma organizada.

GitHub

GitHub permite almacenar proyectos online, compartir código y colaborar con otros desarrolladores.

Lenguajes

Los lenguajes de programación sirven para crear páginas web, videojuegos, aplicaciones y software.

parseInt()

Convierte texto o números decimales en números enteros.

parseFloat()

Convierte texto en números decimales de manera sencilla.

Switch Case

Permite ejecutar diferentes opciones dependiendo del valor de una variable.
